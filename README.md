# Data Science Course Assignment

## Overview
This repository contains my assignment for the **Data Science** course. The project demonstrates fundamental data science concepts, including data exploration, preprocessing, visualization, and machine learning using Python and Jupyter Notebook.

## Objectives
- Understand the dataset
- Perform data cleaning and preprocessing
- Explore data through visualization
- Apply machine learning techniques
- Evaluate model performance
- Draw insights from the results

## Project Files
- `Data_science Assignment_2.ipynb` – Complete Jupyter Notebook containing the assignment.
- `iris.csv` – Dataset used for analysis.

## Tools and Libraries
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## How to Run
1. Clone this repository.
2. Install the required libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
3. Open `Data_science Assignment_2.ipynb` using Jupyter Notebook or VS Code.
4. Run all cells to reproduce the analysis and results.
